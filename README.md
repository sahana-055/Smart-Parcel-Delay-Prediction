# Smart Parcel Delay Prediction

This project predicts whether a parcel delivery will be delayed based on distance, traffic, weather, and delivery type.

## Features
- Simple and fast prediction
- Easy-to-use interface
- Useful for logistics planning

## Tech Stack
- Python
- Streamlit

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Run the application  
   streamlit run app.py
