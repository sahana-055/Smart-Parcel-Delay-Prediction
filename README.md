# Smart Debt Recovery & Case Prioritization System

This project presents a digital solution for managing and prioritizing debt recovery cases using rule-based intelligence.

## Problem Statement
Debt Collection Agencies handle a large number of overdue accounts manually, leading to delays, inefficiencies, and lack of transparency.

## Solution
This system analyzes key parameters such as:
- Outstanding amount
- Payment delay duration
- Customer type
- Payment history

Based on these inputs, it classifies cases into priority levels to help organizations take timely action.

## Features
- Simple and easy-to-use interface
- Priority-based classification
- Supports better recovery planning
- Scalable and adaptable

## Technology Stack
- Python
- Streamlit
- Rule-based logic

## How to Run
1. Install dependencies  
   `pip install streamlit`

2. Run the app  
   `streamlit run app.py`
